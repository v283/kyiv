Дійсні числа

Дійсні числа — це основа математичного аналізу та алгебри. Вони включають всі раціональні та ірраціональні числа. На цій сторінці ви дізнаєтеся про їх властивості, види та приклади використання.

Що таке дійсні числа?

Дійсні числа — це множина чисел, яка включає:

  • Раціональні числа: числа, які можна подати у вигляді дробу a/b, де a та b — цілі числа, b ≠ 0.
  • Ірраціональні числа: числа, які не можна подати у вигляді дробу (наприклад, √2, π).

Дійсні числа позначаються символом ℝ.

Властивості дійсних чисел

Дійсні числа мають такі основні властивості:

  • Коммутативність: a + b = b + a, a * b = b * a.
  • Асоціативність: (a + b) + c = a + (b + c), (a * b) * c = a * (b * c).
  • Дистрибутивність: a * (b + c) = a * b + a * c.
  • Наявність нейтральних елементів: a + 0 = a, a * 1 = a.
  • Наявність обернених елементів: Для кожного a існує -a таке, що a + (-a) = 0.

Види дійсних чисел

Дійсні числа поділяються на кілька видів:

  • Натуральні числа: 1, 2, 3, ...
  • Цілі числа: ..., -2, -1, 0, 1, 2, ...
  • Раціональні числа: числа, які можна подати у вигляді дробу (наприклад, 1/2, -3/4).
  • Ірраціональні числа: числа, які не можна подати у вигляді дробу (наприклад, √2, π).

Приклади використання дійсних чисел

Приклад 1: Розв'язання рівнянь

Розв'яжіть рівняння: 2x + 3 = 7.

Розв'язання: 2x = 7 - 3, 2x = 4, x = 2.

Приклад 2: Обчислення площі

Обчисліть площу кола з радіусом r = 5.

Розв'язання: S = πr² = π * 5² = 25π.

Числова пряма

Дійсні числа можна зобразити на числовій прямій, де кожна точка відповідає певному дійсному числу. Наприклад:

  • Точка 0 — початок відліку.
  • Точка 1 — одиничний відрізок.
  • Точка √2 — ірраціональне число, яке знаходиться між 1.41 та 1.42.

Висновок

Дійсні числа є основою математики та використовуються в різних галузях науки та техніки. Розуміння їх властивостей та видів допомагає у вирішенні складних задач.
