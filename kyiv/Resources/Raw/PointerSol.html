Показникові, логарифмічні рівняння та системи рівнянь

На цій сторінці представлено основні поняття, методи розв'язання та приклади показникових, логарифмічних рівнянь та систем рівнянь.

Показникові рівняння

Показникове рівняння має вигляд ax = b, де a > 0, a ≠ 1.

Метод розв'язання:

  • Якщо b > 0, то рівняння має розв'язок: x = logab.
  • Якщо b ≤ 0, рівняння не має розв'язків.

Приклад 1: Розв'язання показникового рівняння

Розв'яжіть рівняння: 2x = 8.

Розв'язання: x = log28 = 3.

Логарифмічні рівняння

Логарифмічне рівняння має вигляд logax = b, де a > 0, a ≠ 1.

Метод розв'язання:

  • Перетворюємо рівняння до показникового вигляду: x = ab.
  • Перевіряємо, чи задовольняє знайдений корінь області визначення логарифма (x > 0).

Приклад 2: Розв'язання логарифмічного рівняння

Розв'яжіть рівняння: log3x = 4.

Розв'язання: x = 34 = 81.

Системи показникових та логарифмічних рівнянь

Система рівнянь може містити як показникові, так і логарифмічні рівняння. Для їх розв'язання часто використовуються методи підстановки або заміни змінних.

Приклад 3: Розв'язання системи рівнянь

Розв'яжіть систему: 2x = 16, log2y = x.

Розв'язання:

З першого рівняння: 2x = 16, x = 4.

Підставляємо x = 4 у друге рівняння: log2y = 4, y = 24 = 16.

Розв'язок: x = 4, y = 16.

Висновок

Показникові та логарифмічні рівняння та системи рівнянь є важливими елементами математики. Розуміння методів їх розв'язання допомагає у вирішенні складних задач.
